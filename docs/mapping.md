# 🔀 自定义映射

当媒体库里的**番剧名称**和 Bangumi 上的条目对不上时（译名不同、多季合并、特摄/纪录片等），程序可能无法自动找到正确的条目。此时可以在 **「映射管理」** 里为「某个标题」指定对应的 **Bangumi 条目 ID**，同步时会**优先**使用这条规则，避免反复匹配失败。

在 Web 界面左侧打开 **「映射管理」** 即可使用本功能（与「配置管理」同级）。

## 映射里要填什么

每条映射包含两部分：

- **番剧名称**：需要与同步请求里带来的**标题字符串尽量一致**（包括标点、空格、季名等），否则对不上键。若某次同步日志里能看到实际标题，建议按日志里的写法填写。
- **Bangumi ID**：该番在 Bangumi 上的 **subject 编号**（纯数字）。打开对应条目页面，地址一般为 `https://bgm.tv/subject/数字`，其中的数字即为 ID。

程序会按说明使用「**第一季 / 主条目**」对应的 ID：你只需填主条目的 ID，后续季数会由程序按逻辑处理，无需为每一季各写一条（与映射文件内的说明一致）。
